# Expense Tracker

## Overview

Expense Tracker is a web-based personal finance management platform designed to help users efficiently manage their income and expenses.

The application enables users to record financial transactions, monitor account balances, categorize expenses, and track spending habits in real time through a simple and responsive interface.

The platform is designed to make budgeting and financial management easier by providing users with a centralized dashboard for monitoring their financial activities.

## Purpose

The main goal of this project is to:

- Help users track daily income and expenses
- Provide an easy way to monitor financial balance
- Simplify personal budgeting

## Features

- Add income and expense transactions
- View total account balance
- Track total income and expenses
- Categorize transactions
- Search and filter financial records
- View transaction history
- Delete transactions
- Responsive and user-friendly interface

## Installation

To run this project locally:

```bash
git clone https://github.com/your-username/expense-tracker.git
```

```bash
cd expense-tracker
```

Install dependencies:

```bash
npm install
```

Start the React development server:

```bash
npm run dev
```

Start JSON Server:

```bash
npx json-server --watch db.json --port 3001
```
